# ferma
Игра «Симулятор сельского хозяйства»

В этой игре вы создадите собственное мини-хозяйство, выращивая растения на поле 10×10 клеток. Для этого вам понадобятся три инструмента: лопата, ведро с водой и семена.

Начало игры:  
Поле состоит из клеток с «песком» и «камнями». Чтобы посадить растение, нужно выполнить два условия:
1. В клетке нет камня.
2. Клетка вскопана.
Использование инструментов:
- Лопата:  
  — Уберите камень, выбрав лопату и нажав на клетку с камнем.  
  — Вскопайте песчаную клетку, чтобы подготовить её для посадки. Вскопанная клетка отмечена зелёным плюсом.
- Ведро с водой: 
  — Поместите источник воды в любую свободную клетку. Вокруг него образуется область увлажнённой почвы.  
  — Уровень влажности отображается цветом клетки: чем темнее, тем влажнее. Со временем почва высыхает, поэтому для поддержания влажности нужно добавлять воду в источник.
- Семена:  
  — Выберите один из трёх видов растений и посадите его во вскопанную клетку без камня.

Виды растений и их потребности: 
Каждому растению требуется определённый уровень влажности почвы (от 0 до 100):
- Кактус: засухоустойчив, выживает при влажности 0–30
- Картофель: предпочитает среднюю влажность 40–80.
- Болотник: может расти только во влажной среде 70–100.

Растение погибает, если влажность выходит за допустимые пределы. Следите за состоянием почвы и вовремя поливайте её!

Цель игры:  
Успешно выращивать растения, управляя влажностью и правильно используя инструменты. Создайте своё идеальное хозяйство!
